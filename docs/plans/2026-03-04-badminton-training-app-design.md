# 羽毛球专项体能训练计划生成器 - 设计文档

## 概述

一个 PWA 网页应用，帮助羽毛球爱好者（中级以上水平）在健身房进行科学的专项体能训练。基于运动科学论文研究，提供每日变化的训练计划，每个动作配有 MuscleWiki 循环视频演示。

## 用户画像

- 羽毛球中级以上爱好者
- 在健身房训练，器械齐全
- 每次训练 30 分钟
- 训练日：周一、周三、周四、周六
- 主要弱项：核心力量、下肢爆发力、移动速度、体能耐力
- 使用 iOS 手机

## 技术架构

### 技术栈
- 纯前端 PWA：HTML + CSS + JavaScript
- Service Worker：离线缓存 + 通知
- 部署：GitHub Pages（HTTPS，全球可访问）
- 数据存储：localStorage
- 动作演示：MuscleWiki MP4 短视频（autoplay loop muted playsinline）

## 训练内容设计

### 研究依据
基于以下论文：
1. BMC Sports Sci Med Rehabil 2025 - 羽毛球体能训练系统综述（15项研究，370名参与者）
2. PLOS ONE 2024 - 核心力量训练对羽毛球运动员表现的影响
3. PMC 2024 - 核心训练最大化羽毛球表现（机器学习建模）
4. Nature Scientific Reports 2025 - 平衡+增强式训练对变向能力的影响
5. Heliyon 2024 - 增强式训练对羽毛球体能的系统综述

### 核心发现
- 核心是上下肢"能量传导桥梁"，直接影响杀球力量
- 羽毛球是"单腿运动"，需重点练单侧力量
- 旋转爆发力决定杀球威力
- 侧向移动能力比前后移动更关键
- 增强式训练6周可提升敏捷性1.35秒
- 训练应分阶段递进

## 训练计划生成逻辑

### 4天训练日安排
| 训练日 | 重点模块 |
|--------|---------|
| 周一 | 核心旋转力量 + 下肢单侧爆发力 |
| 周三 | 多方向敏捷 + 体能耐力 |
| 周四 | 下肢单侧爆发力 + 肩臂专项 |
| 周六 | 综合日（核心 + 敏捷 + 耐力各一个） |

### 每日计划结构（30分钟）
```
热身       5分钟  (固定动态热身流程)
主训练    20分钟  (3个动作 × 约6-7分钟)
拉伸放松   5分钟  (固定静态拉伸流程)
```

### 周次轮换
- 每个模块6个动作，每天从当日模块中各选1-2个
- 使用日期种子的确定性随机算法，同一天刷新页面计划不变
- 4周一个完整循环，确保每个动作都被练到
- 提供"换一组"按钮，可手动重新生成

## UI/UX 设计

### 页面结构
```
首页（今日训练）
  ├─ 顶部栏：日期 + 训练主题 + 连续打卡天数
  ├─ 热身卡片（可展开查看详细步骤）
  ├─ 动作卡片 ×3（一次显示一个，左右滑切换）
  │    ├─ MuscleWiki 循环视频（侧面+正面）
  │    ├─ 中文动作名 + 羽毛球关联说明
  │    ├─ 组数/次数/休息时间
  │    ├─ 动作要领 + 常见错误
  │    ├─ 倒计时器（大字体，2米外可见）
  │    └─ 完成按钮
  ├─ 拉伸卡片
  └─ 全部完成 → 打卡成功动画

设置页（齿轮图标）
  ├─ 提醒时间设置
  ├─ 训练日选择
  └─ 训练历史/打卡日历
```

### 交互设计
- 移动端优先，大字体大按钮
- 一次只看一个动作卡片，减少信息量
- 计时器字体够大，健身房 2 米外也能看清
- 完成一个动作自动跳到下一个
- 非训练日显示休息提示 + 下次训练预告

### 设计风格
- 主色：#059669（运动感翡翠绿）
- 暗色模式优先（健身房灯光暗更好看）
- 卡片圆角 16px，阴影柔和
- 系统字体栈（PingFang SC 优先）
